 ## Overview
This is a simple dashboard-style web app built using React (Vite) to manage and view a list of companies.
It displays companies using data from a companies.json file and allows users to search, filter, and view details of each company.

## Tech Stack
Component	Technology Used
Frontend	React (Vite)
Routing	React Router DOM
Styling	TailwindCSS
Deployment	Netlify
Data Source	Static JSON file (src/data/companies.json)


 ## Features
 Dashboard-style UI
Company list display (cards/table layout depending on UI choice)
Industry-based filtering (industries intentionally repeated)
Search by company name
Details page for each company using dynamic routing (/company/:id)
Responsive layout

## How to Run Locally

Clone the project or extract the zip
Install dependencies:
npm install



npm run dev
